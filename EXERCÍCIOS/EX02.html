Colocando imagens no site

Para inserir uma imagem precisa inserir a tag img e a foto precisa estar salva dentro da pasta do arquivo do código
Quando colocar a tag IMG apertar ctrl + barra de espaço para que apareça as opções das imagens
Escolher qual foto você quer e colocar texto alternativo para inclusão social

olha só que nenês mais lindos!!! ❤

Para colocar imagens que não estejam salvas dentro da pasta do código, precisa abrir a pasta onde a imagem está salva e selecionar (quando apertar ctrl + barra de espaço.

Também podemos colocar imagens externas
Para isso precisa colocar a TAG img e colar a URL da imagem escolhida
É perigoso isso porque se a imagem cair, ela sai do ar no seu site

Ícones

Usar a extensão .ico para colocar icones

Inserir dentro do head (acima do title), escrever link, esperar aparecer as opções e escolher a favicon e depois com ctrl + barra de espaço escolher a imagem do ícone.
Recomendável que deixe o arquivo renomeado como favicon.ico
