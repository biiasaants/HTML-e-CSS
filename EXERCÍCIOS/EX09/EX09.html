Imagem Dinamica

Usar tag PICTURE, dentro dela inserir uma imagem com IMG normal. Essa imagem precisa ser a de maior resolução. Para a imagem diminuir conforme tela, precisa inserir acima da linha IMG a tag SOURCE (existem vários tipos), selecionar media-type, alterar de min para max, na quantidade de px, colocar um pouco a mais de px para não criar nenhuma barra de rolagem, no item SRCSET incluir a imagem de tamanho MENOR e em TYPE incluir o formato da imagem.
Lembrando que é necessário deixar todas as imagens dentro de uma pasta separada.
A ordem dos tamanhos, precisa ser respeitada.
