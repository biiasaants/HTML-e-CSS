Negrito

Não usar a tag < b > para indicar um negrito, pois na linguagem HTML essa tag altera somente a estética da escrita, ao invés disso usar a tag < strong > que fará o termo ficar FORTE, com sentido semântico para o código.

Texto deletado

Aqui temos um texto deletado tag DEL, que pode ser lido, mas que está com um risco em cima, dificultado a leitura.

Texto inserido (famoso sublinhado)

A tag INS faz com que o texto fique visivel, mas com um risco em baixo, ou seja sublinhado, MAS com sentido semântico, não somente estético. Usado quando queremos que prestem atenção naquela parte do texto.

Texto Sobrescrito

A tag SUP faz com que o texto fique um pouco acima da letra, muito utilizado em equações x2 para teste vou colocar esse texto dentro da tag sup

Texto Subescrito

A tag SUB faz o contrário da sup, ela deixa o texto abaixo da letra x2 para teste vou colocar esse texto dentro da tag sub

Código

Usamos a tag CODE para indicar um código. O navegador irá entender que aquele trecho cita um código, porém a formatação ficará mono-espaçada.
Ao colocar a tag PRE estabelecemos que aquele texto deve seguir a formatação original do HTML.
